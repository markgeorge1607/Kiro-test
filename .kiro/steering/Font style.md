# Font Style Rules

- Use Takeaway Sans Regular (weight 400) for all body copy across the project.
- Use Takeaway Sans Bold (weight 700) for subheadings such as item names, prices, form labels, and secondary labels.
- Body/subheading font stack: `'Takeaway Sans', 'JET Sans Digital', Arial, sans-serif`
- Headings, buttons, and brand elements (JET+ logo text) continue to use JET Sans Digital.
- The `bodyFont` constant should be used for any text at fontWeight 400 or 700 that isn't a primary heading or brand element.
- Use JET Sans Digital 600 for all heading titles
